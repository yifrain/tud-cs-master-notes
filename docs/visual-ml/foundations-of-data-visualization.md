# Data Visualization

## 📖 课程建议

### 2025 课程印象
> Contributor: @yifan | 更新于 2025-03-14

Prof. Stefan Gumhold 线下讲述SciVisIntro之前的内容，其余部分由Prof. Raimund Dachselt，Prof. Gumhold的英语有些口音，
表达不算特别流利，但胜在前半部分课程PPT内容相对容易理解，Prof. Dachselt 上的线上课节奏偏快，但本身内容较难，个人课下要花功夫才能
搞明白，此外本课程 exercise 前几节较简单，但后面会和课程内容相结合出练习题，并且在考试中会专门考察。整体来讲学习压力较大，仅靠上课
是完全不足以应对考试的，opal上有[往年习题和参考习题200+](https://bildungsportal.sachsen.de/opal/auth/RepositoryEntry/50816516096/CourseNode/1724725678549992007)可供参考，实际考试习题量大于样卷，考试中会出现参考习题但仅一小部分按原样考察，如果只以样题为核心复习相关
知识点考试时会很难受。总的来说，除非ML模块其他课程更不感兴趣，否则不建议选这门。

## 📝 考试相关

### 2025 考试分享
> Contributor: @yifan | 更新于 2025-03-14

#### 复习建议

预留足够长的时间，开始可以带着opal上参考习题看PPT复习，exercise 同样重要，每章考察得很均衡，覆盖范围也广，
虽然前半部分内容相对简单也不要忽视后半部分。考试时间一个半小时，试卷题量相当大，不切中要点地写很难做完。此外考试中
对比概念的题目非常多，给你两个概念或者根据某个主题写两个概念，再或者给你三个自己选两个写相关的内容，可以如果只熟悉一部分
只能是看运气拿分。
